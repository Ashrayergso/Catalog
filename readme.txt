# Next.js Equipment Catalog App

This is a modern web application built using the Next.js framework and TypeScript. The application serves as a product catalog for equipment, allowing users to view details, search online, find alternatives, select quantity, and export selected items to an Excel file.

## Tech Stack

- Next.js: A React framework for building modern web applications with server-side rendering and routing.
- TypeScript: A statically typed superset of JavaScript that enhances the developer experience with type checking.

## Features

- Server-side rendering for improved performance.
- Equipment catalog with details such as name, cost, description, manufacturer number, model, type, image, and product link.
- Option to search a selected equipment online.
- Option to search for alternative equipment online.
- User can input a quantity for each item.
- Export selected equipment list to an Excel file.
